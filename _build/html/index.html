ArchlinuxGR Repo documentation!¶

Το παρόν wiki απευθύνεται στους maintainers του ελληνικού Archlinux repository. O κώδικας της σελίδας και των κειμένων μπορεί να βρεθεί στο github. Όποιος θέλει να προσθέσει κάτι μπορεί να στείλει ένα pull request.

Περιεχόμενα:

  • Πώς χτίζουμε σε fakeroot περιβάλλον
  • Φτιάχνοντας την database του τοπικού μας repository
  • Ανεβάζοντας τα χτισμένα πακέτα και τις βάσεις στον remote server
  • Χρήση του git για το tracking των PKGBUILDs
